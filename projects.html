Job Board Website Interncomp

Interncomp is a job board website which makes use of API calling to populate the website with company information, internship oppurtunities as well as user reviews and ratings. A Prisma database was also implemented to allow user login and account creation as well as Google account authentication. From this project I was able to gain full-stack development as well as experience in creating a dockerized environment.

Slash Out!

I worked with a team to create a game called slash out using the unity engine for a student project. Slash Out! Is a game where the player will traverse a 3D environment with 360 degrees of motion. The goal is to avoid projectiles and approach enemies to slash them with a sword which can swing in 8 directions inputted by the user. I created this game in order to understand vector math for 3D environments, creating models in Blender as well as understanding object detection and movement in 3D environments.

Unity Engine C# 3D Game Development 3D Vector Math Blender 3D Movement Game Physics
