Software Developer – Eastern Edge Robotics • January 2024 - Present

Eastern Edge Robotics is a student design team focused on designing and building our own underwater ROV which we use in competition against other university teams in the MATE ROV competition. As a software developer for the team, I contributed to the GUI's development and it's functionality to interact with user and ROV.

I created webpages for settings, ROV commands, and thruster control, along with live ADC and Temperature sensors for the ROV that are displayed on the GUI and updated in real time. I've also created backend software for the joystick & thruster configuration using ROS2 and JSON in a Dockerized environment. This helped immensely since the joystick and thruster's could now be controlled remotely from the GUI. During the competition we were able to operate the ROV from the touch of a button.

My latest task for the team has been the development of OpenCV software to scan data tables of invasive carp species and extract location data. after scanning and storing the data the program will animate the carp spread in the Illinois River based on temporal data visualization.

I am also the electrical engineering presenter and representative for the MATE ROV World Competition where I will present all of the electrical design and implementation on the underwater ROV to a panel of judges followed by a Q&A panel.

ReactJS NodeJS HTML & CSS Python ROS2 Docker Javascript Typescript JSON ReduxJS GUI Data Visualization Embedded Systems OpenCV
